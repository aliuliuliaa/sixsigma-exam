命题趋势（2024-2026）

1. 情景分析题占比上升

单纯理论记忆类题目占比下降，大量被情景分析题替代。典型例题：描述一个对比两种工艺参数的情景，要求判断应使用哪种统计检验方法。

2. 弱化纯数学计算

更侧重对统计分析结果的理解与解读能力。题目会直接给出 Minitab 等软件的分析输出结果，要求考生判定 P 值的意义和结论。

3. DOE 考点繁多

实验设计是近两年重点考查内容，包括全因子/部分因子设计、交互效应分析、中心点设计、效应Pareto图解读等。

4. 多选题成为关键失分点

30 道多选题占 60 分（总分的一半），少选或错选均不得分。对概念边界和逻辑判断要求极高。

DMAIC 五阶段详解

DMAIC 是六西格玛改进现有过程的核心方法论。点击各阶段标题展开/折叠。

D - Define 界定阶段v

目标：明确问题是什么、范围在哪里、目标是什么

  • 识别顾客需求（VOC），转化为CTQ关键质量特性
  • SIPOC图界定过程边界（供应商-输入-过程-输出-顾客）
  • 问题陈述使用Is-Is not分析工具精确定位
  • SMART目标设定（具体、可测量、可实现、相关、有时限）
  • 项目立项表：背景、目标、范围、团队、里程碑
  • QFD质量功能展开：将顾客需求转化为工程指标

关键交付物：项目立项表、SIPOC图、CTQ树

v
M - Measure 测量阶段v

目标：量化当前过程表现，验证测量系统可靠

  • 流程图细化，识别关键输入变量(KPIV)和输出变量(KPOV)
  • 因果图(鱼骨图)和因果矩阵筛选潜在关键因子
  • 数据类型判断：连续型 vs 离散型
  • 抽样计划：确定样本量、抽样方法（随机/分层）
  • 测量系统分析(MSA)：GR&R评估偏倚、线性、稳定性、重复性、再现性
  • 过程能力分析：计算Cp/Cpk/Pp/Ppk，评估DPMO
  • 基线数据收集：量化当前缺陷率、周期时间等

关键交付物：数据收集计划、MSA报告、过程能力基线

v
A - Analyze 分析阶段v

目标：找到影响输出的根本原因和关键因子

  • 探索性数据分析：散点图、箱线图、直方图
  • 假设检验：t检验、方差分析(ANOVA)、卡方检验
  • 非参数检验：Mann-Whitney、Kruskal-Wallis、Wilcoxon
  • 回归分析：线性回归、R-sq与R-sq(adj)评估、残差分析
  • 5 Why分析法和鱼骨图追溯根本原因
  • 价值流图(VSM)分析增值与非增值活动
  • 七大浪费识别：过量生产、等待、运输、过度加工、库存、动作、缺陷

关键交付物：根本原因分析报告、关键因子清单、统计分析报告

v
I - Improve 改进阶段v

目标：确定并实施最优改进方案

  • 头脑风暴、六顶思考帽等创意工具产生解决方案
  • 实验设计(DOE)：全因子2^k、部分因子2^(k-p)、中心点设计
  • 效应Pareto图识别显著因子和交互效应
  • 响应优化器确定最优参数组合
  • 精益工具：5S、看板拉动、SMED快速换型
  • 限制理论(TOC)五步法：识别瓶颈 -> 利用 -> 服从 -> 打破 -> 回到第一步
  • FMEA风险评估：RPN = 严重度 x 发生度 x 检测度
  • 试点试验验证方案效果

关键交付物：改进方案、DOE实验报告、试点结果

v
C - Control 控制阶段v

目标：固化改进成果，防止过程回退

  • 制定控制计划：控制对象、规格、方法、频率、反应计划
  • SPC统计过程控制：选择合适的控制图类型
  • 计量型数据：Xbar-R图、Xbar-S图、I-MR图
  • 计数型数据：p图、np图、c图、u图
  • Nelson判异规则：8条判异准则识别异常模式
  • 防错技术(Poka-Yoke)：从源头设计使错误不可能发生
  • 更新SOP和作业指导书，标准化操作
  • 可视化管理看板、项目经验文件化
  • 培训计划制定，建立持续监控机制

关键交付物：控制计划、SPC控制图、更新后的SOP、培训计划

核心公式速记表
过程能力
Cp (过程能力指数)
Cp = (USL - LSL) / (6 * sigma)
仅考虑规格宽度和过程变异，不考虑均值偏移
Cpk (过程能力指数，考虑偏移)
Cpk = min[(USL - mean) / (3*sigma), (mean - LSL) / (3*sigma)]
取单侧最小值，反映最不利方向的能力
Pp / Ppk (过程绩效指数)
Pp = (USL - LSL) / (6 * S)
S为总体标准差(长期)，Cp/Cpk用子组内标准差(短期)
DPMO (每百万机会缺陷数)
DPMO = (缺陷数 / (产品数 * 机会数)) * 1000000
3.4 DPMO = 6 sigma水平(考虑1.5sigma偏移)
假设检验
P值判定规则
P < alpha => 拒绝H0 (差异显著)
P >= alpha => 不拒绝H0 (差异不显著)
alpha通常取0.05。第一类错误=弃真=P(alpha)，第二类错误=取伪=P(beta)
t统计量
t = (xbar - mu0) / (s / sqrt(n))
单样本t检验：比较样本均值与目标值
检验功效 (Power)
Power = 1 - beta
beta为第二类错误概率。增大样本量可提高功效
方差分析
F统计量
F = MS(组间) / MS(组内)
F大 => 组间差异显著 => P小 => 拒绝H0(各组均值相等)
自由度
df(总) = N - 1; df(组间) = k - 1; df(组内) = N - k
N=总观测数，k=组数
回归分析
R-sq (决定系数)
R-sq = SS(回归) / SS(总) = 1 - SS(残差)/SS(总)
范围[0,1]，越接近1拟合越好。但增加变量总使R-sq增大
R-sq(adj) (调整决定系数)
R-sq(adj) = 1 - (SS残/(n-p-1)) / (SS总/(n-1))
惩罚不必要变量。R-sq与R-sq(adj)差距大 => 含不显著变量
相关系数 r
r = Cov(X,Y) / (sigmaX * sigmaY)
范围[-1,1]。|r|>0.8强相关，但相关不等于因果
实验设计 DOE
全因子实验次数
实验次数 = L^k (L=水平数, k=因子数)
2水平3因子: 2^3=8次; 2水平4因子: 2^4=16次
部分因子实验次数
实验次数 = L^(k-p) (p=分数指数)
2^(3-1)=4次(半分式); 2^(5-2)=8次(四分之一分式)
控制图
控制限设置
UCL = CL + 3*sigma; LCL = CL - 3*sigma
3sigma原则: 稳态下99.73%数据在控制限内
Xbar-R图参数
CL(Xbar) = Xbarbar; UCL(Xbar) = Xbarbar + A2*Rbar
CL(R) = Rbar; UCL(R) = D4*Rbar; LCL(R) = D3*Rbar
A2/D3/D4是随子组大小n变化的常数
p图控制限
CL = pbar; UCL = pbar + 3*sqrt(pbar*(1-pbar)/n)
pbar=平均不合格率, n=样本量
FMEA
RPN (风险优先数)
RPN = S(严重度) * O(发生度) * D(检测度)
每项1-10分，RPN范围1-1000。注意：检测度越低(越难检测)分数越高
财务评估
投资回收期
回收期 = 投资额 / 年节约额
一般2年以内为有价值项目
ROI (投资回报率)
ROI = (收益 - 投资) / 投资 * 100%
NPV>0值得投资，NPV<0不值得
精益
节拍时间 (Takt Time)
Takt Time = 可用工作时间 / 客户需求量
决定了生产节奏，是VSM分析的核心指标
OEE (设备综合效率)
OEE = 可用率 * 表现率 * 质量率
世界级OEE目标>85%
正态分布
经验法则
mean+/-1sigma = 68.27%
mean+/-2sigma = 95.45%
mean+/-3sigma = 99.73%
mean+/-6sigma = 99.99966% (3.4 PPM)
3.4PPM是考虑了1.5sigma均值偏移后的结果
田口方法
信噪比 S/N (望目特性)
S/N = 10 * log10(mean^2 / variance)
S/N越大表示产品质量越稳健、变异越小
备考策略与应试技巧
核心备考原则
1. 理解重于记忆：中质协考试近年弱化纯计算，强化对统计结果的理解和解读。不要只会套公式，要理解「为什么用这个工具」「P值意味着什么」「F值大说明什么」。
2. 多选题是胜负手：60分的多选题（30题x2分），少选或错选均不得分。多选题的难度远高于单选题，必须系统训练概念边界和逻辑判断能力。
3. 情景题是趋势：大量题目以实际案例形式出现，要求判断「应该使用哪种统计方法」「分析结果说明什么」。重点是训练工具选择能力，而非计算能力。

答题策略

时间管理（90题/180分钟 = 每题2分钟）

单选题先做、快速过；多选题慢做、仔细审。建议前60道单选题用60-70分钟完成，后30道多选题用90-100分钟。留10-15分钟检查。

多选题技巧

  • >> 先判断「确定正确」的选项，再看其他选项
  • >> 注意「绝对」「一定」「所有」等极端表述（通常为错）
  • >> 概念边界题：区分「属于」和「不属于」的类别
  • >> 不确定时宁可不选（漏选不得分但也不会扣分，错选全扣）
  • >> 实际上：少选不得分！所以如果确定2个是对的但不确定第3个，需要权衡

情景分析题技巧

  • >> 先看问什么（选择工具？解读结果？判断结论？）
  • >> 提取关键信息：数据类型、组数、是否正态、是否等方差
  • >> 数据类型 -> 决定参数/非参数检验
  • >> 组数 -> 2组用t检验，>2组用ANOVA
  • >> 给出Minitab输出 -> 看P值判断结论
错题管理方法

错题本建立标准

  • >> 按知识点分类（如：假设检验、DOE、SPC等）
  • >> 标注错误原因：公式记忆混淆 / 流程理解有误 / 概念边界不清 / 读题失误
  • >> 每周回顾一次错题本，重点看反复出错的题型
  • >> 考前1周不做新题，专攻错题本
注意：中质协考试是闭卷，不能带任何参考资料入场（与ASQ开卷不同）。所有公式必须记住。建议考前制作公式速查卡片反复记忆。
